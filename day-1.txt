phases in git

command to clone repository ==> git clone repo-url

working to staging/index ==> git add filename (for specific file name) or git add . (for select all
files)

staging/index to local repository ==> git commit -m " commit message"

local repository to central repository ==> git push

creating a branch ==> git branch branch-name

check the branch list ==> git branch

checkout to creating branch ==> git checkout branch-name

create branch and checkout at the same time ==> git checkout -b branch-name
